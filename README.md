# 况两把 · 猜球员

一款受 [Wordle](https://www.nytimes.com/games/wordle) 启发的足球球员猜测游戏，玩家需要在有限次数和时间内猜出神秘球员。

## 在线体验

GitHub Pages: `https://yefei404.github.io/player_guess/`

## 游戏规则

- 系统随机选择一名球员作为答案
- 玩家有 **6 次猜测机会** 和 **200 秒** 时间限制
- 每次猜测后，系统对以下 6 个维度给出颜色反馈：

| 维度 | 🟩 完全正确 | 🟨 接近 | ⬛ 错误 |
|------|-----------|---------|---------|
| 身高 | 完全一致 | 差值 ≤5cm（含箭头提示高/矮） | 差值 >5cm |
| 号码 | 完全一致 | 差值 ≤5（含箭头提示大/小） | 差值 >5 |
| 位置 | 完全一致 | — | 不同 |
| 球队 | 完全一致 | — | 不同 |
| 国籍 | 完全一致 | — | 不同 |
| 状态 | 完全一致 | — | 不同 |

## 功能特性

- **搜索补全**：支持中文名和英文名模糊搜索，带权重排序，支持键盘方向键导航选择
- **球员库浏览**：弹窗展示全部球员，支持按位置、状态、国籍、身高、号码筛选和排序
- **实时倒计时**：基于 `Date.now()` 防漂移计时，进度条颜色随剩余时间变化（绿 → 黄 → 红）
- **游戏战绩**：基于 localStorage 持久化记录，统计已玩局数、胜率、连胜、猜测分布，支持一键清空
- **动画反馈**：猜错时行抖动（shake），猜对时弹跳（pop），新行滑入（slideIn）
- **响应式布局**：适配桌面端和移动端，移动端显示横向滑动提示

## 技术方案

### 技术栈

| 层面 | 方案 |
|------|------|
| 结构 | 原生 HTML5 |
| 样式 | 原生 CSS3（CSS Variables 主题系统） |
| 逻辑 | 原生 JavaScript（ES6+） |
| 数据 | 独立 JS 文件，按分类整理 |
| 持久化 | localStorage（游戏战绩） |
| 部署 | GitHub Pages（静态托管） |

### 球员数据模型

每位球员包含以下字段：

```javascript
{
  name: "梅西",              // 中文名
  nameEN: "Lionel Messi",   // 英文名
  height: 170,              // 身高 (cm)
  position: "FW",           // 位置: GK/DEF/MID/FW
  team: "迈阿密国际",        // 当前球队（退役球员为 "退役"）
  nationality: "阿根廷",     // 国籍
  status: "现役",            // 状态: 现役/退役
  number: 10                // 球衣号码
}
```

### 核心模块

| 模块 | 职责 |
|------|------|
| 游戏状态管理 | `startNewGame()` 初始化、`submitGuess()` 提交、`onTimeout()` 超时处理 |
| 搜索引擎 | `matchScore()` 多级权重匹配（精确 > 包含 > 前缀 > 部分），`norm()` 统一化处理 |
| 键盘导航 | 方向键选择补全项，Enter 确认，Escape 关闭 |
| 反馈系统 | `cmpH()` 身高比较、`cmpN()` 号码比较、`cmpE()` 精确比较 |
| 战绩系统 | `loadStats()` / `saveStats()` / `recordResult()` / `clearAllStats()` |
| 球员库弹窗 | `renderModal()` 多维筛选 + 排序渲染 |
| 计时器 | 基于 `Date.now()` 差值的防漂移计时 |

### 战绩数据结构（localStorage）

```javascript
{
  played: 12,          // 已玩局数
  won: 9,              // 胜利局数
  currentStreak: 3,    // 当前连胜
  maxStreak: 5,        // 最佳连胜
  dist: [5,3,2,1,1,0]  // 猜测分布（第1~6次猜中的次数）
}
```

## 本地运行

无需安装依赖，直接用浏览器打开即可：

```bash
open index.html
# 或使用任意静态服务器
npx serve .
```
